Este projeto se trata de um visualizador de estruturas de dados, desenvolvido com o framework JavaFX e auxílio da ferramenta Scene Builder para a construção da interface gráfica do usuário.

Quanto a features:
-Suporte a visualização de listas ligadas (simples, duplas e circular);
-Suporte a visualização de pilhas e filas;
-Operações de criação de estruturas, inserção, remoção e busca;
-Botões para fazer e desfazer o passo a passo das animações;
-Barra de progresso para acompanhar o andamento da execução.

Quanto ao desenvolvimento:
-Código fonte escrito com a linguagem Java;
-JavaFX foi o framework utilizado;
-Folhas de estilo em cascata (CSS) foram utilizadas para facilitar na construção do visual da inteface gráfica;
-Scene Builder foi uma ferramenta utilizada (própria para o JavaFX) na construção da interface estática da aplicação.

É o produto do meu Trabalho de Conslusão de Curso (TCC) do curso de Bacharelado em Ciência da Computação no IFMA Campus Caxias.
